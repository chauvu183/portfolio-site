REDESIGN · KFZ MODULE

eVB-Suche Redesign

Angular · REST APIs · .NET (print engine) · End-to-end
THE CHALLENGE

A core broker task, buried in a clunky legacy search

The eVB number is what a broker needs to register a vehicle — so finding one fast matters. But the legacy eVB search made it hard: results were thin and unclear, error messages from the API arrived split into confusing fragments, and there was no insight into how the feature was used. Insurers also needed a way for back-office staff to research eVB numbers without touching tariff settings — an access path that didn't exist yet.

IMPACT AT A GLANCE
Owned end-to-end
Concept → UI → API → AI-analyzed analytics · a build → measure → learn loop
BEFORE → AFTER

  • × No usage analytics
  • × No back-office research path
→
AFTER · REDESIGNED

  • ✓ Quick & detail search modes
